# TaskTracker

Консольное приложение для управления задачами и ошибками (Bug / Story) с автоматическим сохранением данных в файл.

---

## 📋 Описание

TaskTracker — это консольный трекер задач, который позволяет:
- Изменять статус задач (NEW → IN_PROGRESS → DONE)
- Просматривать все добавленные задачи
- Автоматически сохранять данные в файл при выходе
- Автоматически загружать данные при запуске
  
## Технологии
- Java Core
- OOP (наследование, абстрактные классы, полиморфизм)
- Сериализация (ObjectOutputStream / ObjectInputStream)
- Collections Framework (хранение задач в ArrayList)

## Как запустить
1. Склонировать репозиторий
2. Открыть в IntelliJ IDEA
3. Запустить main() в Main.java

## Функции
- Добавление Bug / Story
- Изменение статуса (NEW, IN_PROGRESS, DONE)
- Просмотр всех задач
- Автосохранение в файл tasks.dat
